# 파란기린

파란기린은 카이스트 친구들이 모여 구상한 사이드 프로젝트로, 질문과 함께하는 아이 성장앨범 어플입니다.

[앱스토어](https://apps.apple.com/kr/app/파란기린/id1571656948)와 [플레이스토어](https://play.google.com/store/apps/details?id=com.dandi.parangirin)에 출시한 상태이지만, 지속적인 유료 서비스(naver clova api 등)의 필요로 현재 서비스는 중단된 상태입니다.

### 메인페이지
- 매일 바뀌는 '오늘의 질문'과 다른 친구들의 답변을 확인할 수 있는 '생각 뽐내기'로 구성되어 있습니다.   
- '오늘의 질문'을 확인하고 싶다면 상단의 파란기린을 터치하면 됩니다. 
  - 각 질문마다 스토리와 가이드가 제공되어 있어서 아이들이 대답하기 어려워할 경우 참고하면 좋답니다 :)   
  - 파란기린과 대화하기를 누르면, 파란기린이 음성으로 질문을 합니다. 
  - 질문이 끝나면, 빨간 레코드 버튼을 눌러서 아이의 대답을 담은 영상을 촬영할 수 있습니다.
- '생각 뽐내기'에 있는 다른 친구들의 답변을 보고 파란기린과 대화하는 방식을 파악해보세요.
  - 우리 아이 뽐내기를 누르면 우리 아이의 대화 영상도 생각 뽐내기에 올라갈 수 있어요!

### 질문 / 앨범 / 마이페이지
- 질문 페이지에는 다섯 가지 카테고리로 분류된 질문들이 있습니다.
  - 단, 파란기린과의 대화는 하루에 한 번만 가능해요.
- 캘린더 페이지에서는 파란기린과 대화한 날짜에 녹화한 영상을 확인할 수 있습니다. 
- 마이페이지에서는 다양한 설정이 가능합니다.
  - 프로필 사진 설정: 카메라 버튼을 누르면 설정할 수 있어요.
  - 자녀관리: 여러 자녀를 한 번에 관리할 수 있어요.
  - 의견 보내기: 파란기린팀에게 창의적인 질문을 추천하거나 다양한 의견을 제시할 수 있어요.
  - 로그아웃
  - 그 외에도 파란기린 소개, 공지사항, FAQ, 이용약관, 개인정보 취급방침을 확인할 수 있어요.

## 기술 스택!



- Flutter: 크로스 플랫폼이라는 장점을 활용해서 안드로이드와 ios 앱 개발을 동시에 진행하였습니다.
- firebase: 사용자 데이터와 로그 수집, auth를 관리합니다.
